# Tiny NoirJS app

This repo contains the full code from [this](https://noir-lang.org/docs/tutorials/noirjs_app) Noir docs page.

## Noir project

Uses `nargo` version 1.0.0-beta.15.

Recompile with

```bash
nargo compile
```

## Vite project

Install dependencies with

```bash
yarn install
```

Run app with:

```bash
yarn vite dev
```
